# 大良鱼灯数字工艺体验：粒子 + 工艺影像版

双击运行 `start.ps1`，然后在浏览器打开 `http://localhost:4174`。

原始粒子版仍然保留在 `http://localhost:4173`，方便并排对比。

## 新增内容

- 右上角加入 `CRAFT VIDEO / 工艺影像` 辅助窗口。
- 视频自动播放、静音、循环，并保持内联展示。
- 点击左侧 `01` 至 `04` 或按键盘数字键切换工艺时，视频会同步切换并从头播放。
- 视频加载失败时会显示 `视频素材待导入 / VIDEO PENDING`。
- 左下角加入 `GESTURE SENSOR` 摄像头手势模块。
- 点击 `开启手势识别` 后，浏览器会申请摄像头权限并加载本地 MediaPipe 手势模型。

## 手势

- 食指与拇指稳定捏合 `PINCH`：画模描线。
- 剪刀手 `Victory`：熏竹扎架。
- 先握拳 `Closed_Fist` 再张开手掌 `Open_Palm`：扪纱描鳞。
- 拇指与食指形成圆圈的 `OK` 手势：着色点睛。

识别逻辑包含连续帧稳定判断，避免频繁误切换。MediaPipe 模型保存在 `models/gesture_recognizer.task`。

每一步手势识别成功后，鱼灯粒子会聚合显形并停留 20 秒，方便现场讲解。最后 1 秒会提示准备下一动作，随后自动进入下一步。第四步完成后停留在完成状态。

现场调试时也可以按空格键，模拟当前步骤手势通过后的粒子显形效果。

## 操作

- 点击左侧 `01` 至 `04` 切换工艺环节。
- 使用键盘数字键 `1` 至 `4` 切换。
- 拖动右侧鱼灯改变观看角度。
- 使用右下角面板实时调节粒子密度、粒径、辉光、微扰和旋转速度。
